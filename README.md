Main class: Home.java

Data files needed: 
- Stage JSONs
- Keybinding JSONs
- Music and sound files

- CSS files for styling

- 2 game modes - stamina, and stocks
- 4 Stages
- User preferances with keybindings
- Networked games
- Dark mode
